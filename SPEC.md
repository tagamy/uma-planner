# UMA Planner 仕様書

## 初回リリース
### 概要
育成進捗を管理し、目標をトラッキングするWebアプリ「UMA Planner」を作成する。

### 機能
- 編成したサポートカードの連続イベントの進行度（全3段階）を管理する
  - 進行度はゲーム内UI（「＞」が3つ並んだインジケーター）を模倣し、タップ等で段階を進められるようにする
  - 選択肢によってイベントが終了した場合の「打ち切り（終了）」状態も管理できるようにする
  - サポートカードは5枠（※友人枠などは進行を制御できるため管理対象外）
  - サポートカードは以下の項目を編集できる
    - カード名 (テキスト入力)
    - タイプ (ドロップダウン選択)
      - スピード
      - スタミナ
      - パワー
      - 根性
      - 賢さ
- 切れ者イベントが発生した (通常イベントと衣装イベントの2種類)
　- さらに切れ者ステータスを取得できた
- 適正Sを継承した
  - 距離適正
  - バ場適正
  - 脚質適正
- 任意のスキルを継承できた (スキル名を入力して項目を追加できる)
  - レースの真髄：体
  - レースの真髄：力
  - レースの真髄：速
- 初期状態にリセットできる
    
### 要件
- アプリはWebブラウザで操作できる
  - スマートフォンからの利用を想定し、レスポンシブデザインとする
  - PWA（Progressive Web App）に対応し、ホーム画面に追加してアプリのように利用できるようにする
- CloudFront + Amazon S3 で配信できる（静的サイト）
- 編集した内容はブラウザ（LocalStorage）に記録して、常に最新状態が維持される
  - セーブスロットは分けず、1プレイ分のみ保存する

### 要求
- UIはゲームウマ娘のUIを模倣する
  - ウマ娘のゲームプレイ中（縦画面）に、スマホを横に置いて片手で操作しやすいレイアウトにする

## 機能追加 2026-02-28
###　新シナリオ「Beyond Dreams」対応

### 機能
- 半年ごとの作成会議で強化するトレーニングのLV目標を設定できる
 - トレーニングの種類
   - フィジカル
   - テクニック
   - メンタル
 - LVの範囲
   - 1から8まで
 - 作戦会議の回数
   - 6回
 - 作戦会議のタイミング
   - ジュニア開始
   - ジュニアデビュー前
   - ジュニア12月後半
   - クラシック合宿前
   - クラシック12月後半
   - シニア合宿前
 - LV目標設定例(フィジカル, テクニック, スキル)
  - ステータス重視
   - 311
   - 313
   - 425
   - 545
   - 548
   - 758
  - スキル重視
   - 311
   - 313
   - 425
   - 455
   - 458
   - 758
- プレイヤーはウマ娘を育成しながら、設定した目標をにあわせてDPを振ることができる
